# PicDropNFC Themes

All themes keep the same layout and only change visual style. The default full implementation is `theme-01`.

1. **theme-01 — Wedding Classic Elegant** (fully implemented): lavender + ivory palette, soft glow, floating particles.
2. **theme-02 — Wedding Rustic Barn**: warm neutrals, rosewood tones, cozy gradient drift.
3. **theme-03 — Wedding Modern Minimal**: crisp monochrome base with cool accent gradients.
4. **theme-04 — Baby Shower / Newborn**: airy pastels, soft cloud-like highlights.
5. **theme-05 — Graduation**: confident jewel tones with celebratory shimmer.
6. **theme-06 — Birthday Party**: playful saturated colors and light confetti dots.
7. **theme-07 — Celebration of Life / Memorial**: calm muted palette, respectful subtle movement.
8. **theme-08 — Corporate Event / Conference**: polished blue-slate look, professional and clean.
9. **theme-09 — Holiday / Family Gathering**: cozy seasonal warmth with subtle sparkle.
10. **theme-10 — Sports / Tailgate**: energetic contrast colors and dynamic gradient motion.

To complete any stub theme, tune CSS custom properties and background layers in each `theme.css`.
